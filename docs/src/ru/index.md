# Руководство IPAnalyzer

## Краткое введение

* **IPAnalyzer** — это свободное программное обеспечение под лицензией MIT, которое преобразует двумерные изображения порошковой дифракции (кольца Дебая–Шеррера), записанные с помощью imaging plate (IP) или плоскопанельных детекторов CCD/CMOS, в высокоточные одномерные профили 2θ–интенсивность.
* Программа калибрует геометрию измерения — длину камеры, длину волны, наклон детектора и форму пикселя — по кольцам эталонных материалов, поддерживает рентгеновские, электронные и нейтронные источники и интегрируется с [PDIndexer](https://github.com/seto77/PDIndexer) для последующего анализа пиков.
* Её устройство и возможности следуют *PIP*, разработанному Hiroshi Fujihisa в AIST.

## Возможности

* **Широкая поддержка форматов** : Fuji BAS2000/2500/FDL, Rigaku R-AXIS IV/V, ITEX, Bruker CCD, Rayonix, MAR research, Perkin Elmer, ADSC, RadIcon, Rad-Xcam, HDF5/NeXus, Digital Micrograph 3/4 и общие форматы изображений. Большинство операций ввода-вывода файлов поддерживают перетаскивание (drag & drop).
* **Преобразование изображения в профиль** : Концентрическое (радиально усреднённое), радиальное (азимутальное/cake) и развёрнутое вычисление изображения с использованием субпиксельного алгоритма распределения площади, который корректно обрабатывает наклон детектора и параллелограммные формы пикселей.
* **Калибровка геометрии** : Геометрическое (double-cassette) уточнение длины волны, длины камеры, размера/искажения пикселя и наклона (φ, τ), а также поиск по сетке полным перебором для трудных данных.
* **Работа с изображениями** : Автоматическое определение центра пучка, обнаружение и маскирование рефлексов монокристалла (spot), окружное размытие, наложение колец, калибровка детектора по таблице интенсивностей и сохранение IPA с сохранением геометрии.
* **Многокадровые данные** : Выбор, усреднение или суммирование кадров HDF5/NeXus и других последовательных данных, с длиной волны для каждого кадра из встроенной энергии.
* **Автоматизация** : Auto Procedure с отслеживанием папки и [макрос](5-macro/index.md) с синтаксисом Python для пакетной и сценарной обработки.
* **Интеграция с PDIndexer** : Отправка профилей в [PDIndexer](https://github.com/seto77/PDIndexer) через буфер обмена и запуск там именованных макросов.
* **Светлая / тёмная тема** : Интерфейс следует выбираемому светлому или тёмному цветовому режиму.

## Быстрый старт

1. Скачайте и установите из [Releases](https://github.com/seto77/IPAnalyzer/releases/latest).
2. Считайте дифракционное изображение (File → Read image или перетаскиванием).
3. Задайте **Wave source** и **Detector condition** (длина камеры, размер пикселя, приблизительный центр) в окне свойств.
4. Найдите центр, при необходимости замаскируйте рефлексы и нажмите **Get Profile**, чтобы получить 1D-профиль.
5. Если геометрия неизвестна, откалибруйте её по эталонному материалу — см. [Процедуры](4-procedures.md).

Полный рабочий процесс см. в разделе [Процедуры](4-procedures.md).
