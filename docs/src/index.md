# IPAnalyzer Manual

IPAnalyzer processes Debye-ring patterns collected by flat panel detectors and converts diffraction images into profiles for calibration and analysis.

This GitHub Pages manual is being prepared as the successor to the legacy Word/HTML/PDF manual in `IPAnalyzer/doc/`.
